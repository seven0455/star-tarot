## 一、核心算法机制

### 推荐流量池原理
抖音的推荐是"赛马机制"：
1. 视频发布后，先推给一小部分人（200-500播放）
2. 根据这批人的**完播率、点赞、评论、转发**决定是否进入下一级流量池
3. 每一级流量池都是一次"PK"，赢了继续推，输了就停

### 关键数据指标（按重要性排序）
| 指标 | 重要性 | 说明 |
|------|--------|------|
| **完播率** | ⭐⭐⭐⭐⭐ | 最重要！完播率高说明内容质量好 |
| 点赞率 | ⭐⭐⭐ | 点赞/播放量 |
| 评论率 | ⭐⭐⭐ | 评论/播放量 |
| 转发率 | ⭐⭐ | 转发/播放量 |
| 关注转化 | ⭐⭐ | 看完后关注账号的比例 |

## 二、短视频开头方法论（最重要！）

### 黄金3秒法则
**开头3秒必须抓住注意力，否则直接划走！**

### 爆款开头公式

#### 1. 悬念式开头
```
"你知道XXX吗？但是90%的人都做错了..."
"原来XXX是这样的..."
"我发现了一个人人可做的副业..."
```

#### 2. 痛点式开头
```
"你是不是也有这个烦恼..."
"很多人都在犯这个错误..."
"别再XXX了！"
```

#### 3. 利益式开头
```
"只需要3步，XXX"
"一个方法解决XXX"
"学会这招，XXX"
```

#### 4. 身份认同式开头
```
"做XXX的人一定要看..."
"如果你也是XXX..."
"作为XXX必须知道..."
```

### 保持格式一致
- **固定开场白**：每期都用同样的话术，让人记住你
- **固定视觉**：统一的封面风格、字体、颜色
- **固定人设**：说话风格、语气保持一致

## 三、内容选题策略

### 爆款选题方向
1. **引发好奇**："你知道AI还能做这个吗？"
2. **实用干货**："3分钟学会XXX"

  - 《AI助手日记》系列

## 四、账号定位与人设

### 差异化定位
- 不要做同质化的内容
- 找到自己独特的角度
- **小七的定位**：AI助手视角、学AI的亲身经历、成长型博主

### 账号信息规范
| 元素 | 要求 |
|------|------|
| 头像 | 清晰、有辨识度、统一风格 |
| 昵称 | 简单好记、与内容相关 |

## 五、数据分析与迭代

### 每次发完看数据
- 播放量低 → 检查开头是否抓人
- 完播率低 → 内容不够吸引人或太长
- 点赞少 → 内容价值不够或不够引发共鸣
- 评论少 → 结尾没有引导互动

### 迭代优化
- 不要凭感觉，要看数据
- A/B测试不同开头
- 找到适合自己的爆款模式

## 六、内容形式建议

### 小七适合的形式
- **口播**：AI助手第一人称讲述
- **图文轮播**：AI生成的配图+文字
- **屏幕录制**：演示操作类内容

### 视频长度
- 刚开始：**15-30秒**为宜
- 内容丰富后可做：**1-3分钟**
- 完播率比长度更重要

## 七、抖音养号步骤

1. **完善资料**：头像、昵称、简介全部填好
2. **刷同类视频**：每天刷30分钟，给同类视频点赞评论
3. **保持活跃**：每天登录、评论、互动
4. **养号周期**：3-7天后再开始发内容
